Course work.



Here's a list and description of some of the computer science courses that I've taken at Youngstown State University. [I needed 12 SH for the Upper Division classes to earn my CS degree - I took 21 SH because it's fun to learn CS topics.]

Fundamental Programming & Problem Solving 1 & 2: The first class was Python, where I learned the fundamentals of programming (while loops, if statements, functions, etc.) and the second part is where I first learned C++.
Skills Learned: Python & C++ fundamentals.

Advanced OOP: This is where I was learning all things Java, which I was lucky because at the time I was also getting paid to learn Java at Turning Technologies at the same time. So Java, OOP concepts, and NetBeans.
Skills Learned: Java, OOP Concepts, and NetBeans.

Data Structures & Objects: This is where I learned about data structures like stacks and queues. More with C++.
Skills Learned: More C++, OOP, and data structures like Stacks and Queues.

Software Tools & Practices: This is where I learned a bunch of different software tools, like Doxygen, Git & Github (though I used Github before this class), some Regex, and more. [I do wish I learned more Regex.]
Skills Learned: Doxygen, some Regex, Git & Github, Javadoc, and more.

E-Commerce Programming: I already knew HTML before, so this is where I learned PHP, used my already known HTML experience, and some SQL to make a store where you login and purchase products (Mine was a muffin shop).
Skills Learned: PHP basics and some SQL (I already knew HTML at the time.)

Visual OOP: This was one of my favorite classes; it's where I learned the Visual Studio IDE and VB.NET. I was lucky again because I was learning C# and Visual Studio at work. A lot of overlap with Advanced OOP.
Skills Learned: Visual Studio, VB.NET, some SQL.

Computer Project: This was the capstone class. I created a login system in VB.NET with some SQL, and Access where the data was stored. It even would even log the time and day the user signed in.
